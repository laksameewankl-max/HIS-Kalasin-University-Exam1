# 🏥 ระบบสารสนเทศโรงพยาบาลมหาวิทยาลัยกาฬสินธุ์ (HIS)

โปรเจกต์นี้เป็นส่วนหนึ่งของข้อสอบปฏิบัติ โดยแบ่งออกเป็น 2 ส่วนหลัก:

## 1. ฐานข้อมูลและการสืบค้น (SQL Query)
- **ไฟล์:** `database_setup.sql`
- **คำอธิบาย:** ประกอบด้วยการสร้างตาราง `Patient_Data`, `Visit_Log`, และ `Diagnosis_Record` พร้อมชุดข้อมูลตัวอย่าง 5 รายการ และคำสั่ง SQL สำหรับดึงข้อมูลผู้ป่วยเบาหวาน (E11) ช่วงเดือน ม.ค. - มี.ค. 2026 ของคลินิกอายุรกรรม

## 2. การออกแบบหน้าจอ (UI/UX Design)
### แนวคิด "EMR Teaching Dashboard"
เพื่อให้ตอบโจทย์โรงพยาบาลมหาวิทยาลัยที่เน้นการเรียนการสอนควบคู่กับการรักษา:
- **Split-Screen Workflow:** หน้าจอแบ่งส่วนการบันทึกของนักศึกษาแพทย์ (ซ้าย) และส่วนการตรวจรับรองของอาจารย์แพทย์ (ขวา) เพื่อให้เกิดการ Review งานได้ทันที
- **Patient Safety First:** แสดงแถบ **"แพ้ยา (Drug Allergy)"** เป็นสีแดงเด่นชัดที่ส่วนบนสุดของหน้าจอ เพื่อลด Medication Error
- **Efficiency:** มีระบบ **Quick Order Set** สำหรับโรคเรื้อรัง (เช่น เบาหวาน E11) เพื่อลดจำนวนคลิกและป้องกันการสั่งยาผิดพลาด
- **Information Hierarchy:** แสดงประวัติโรคประจำตัวและผลแล็บสำคัญไว้ในหน้าหลัก เพื่อให้แพทย์เห็นภาพรวมโดยไม่ต้องกดสลับหน้าบ่อยครั้ง
